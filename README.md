# wordpress-sucks
A suck ass wordpress setup

## Get Started

#### Installation
```
$ git clone https://github.com/samuelngs/wordpress-sucks.git
$ cd wordpress-sucks
```

#### Run development
```
$ make up
```

#### Stop server
```
$ make stop
```

#### Remove everything
```
$ make rm
```
